Welcome to AutoChannel

What is AutoChannel?

AutoChannel is desinged to make managing voice channels much easier. This allows Admins/High ranking moderators to manage which categories are enabled to have voice channels popualated.

This is done by enabling autochannel for a specified category and setting unique "prefix". The prefix by default will use "AC!". It is recommended that the prefix never overlap none AutoChannels as the bot will think those channels also belong to it.

Currently AutoChannel bot will always keep 1 empty channel free. This means it will scale up and down if you join an auto generated channel in an enabled category

How to use?

  1. Login on left navigation panel
  2. select a guild (discord lingo or "server")
  3. Accept/add bot to your "guild"
  4. click the dashboard link
  5. Toggle the categories you want enabled
  6. Updated the prefixes and click the apply Checkmark only for updating prefixes
  7. in your sever as an admin run !acupdate
